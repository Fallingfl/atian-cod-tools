Tools to enable EEs in Black Ops 4 and Cold War for custom mutation, casual or offline.

## Black Ops 4 PC

```pwsh
acts mod t8cee
```

## Black Ops 4 PS4

See [ps4](ps4.md) to know how to use the ps4 integration.

```pwsh
acts ps4t8cee [ip:port]
```
